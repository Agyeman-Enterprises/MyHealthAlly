# MyHealthAlly Android App

Jetpack Compose Android app for MyHealthAlly continuous care platform.

## Architecture

- **MVVM + Clean Architecture**
- **Jetpack Compose** for UI
- **Retrofit** for networking
- **Room** for local caching
- **WorkManager** for background sync
- **Health Connect** for health data integration

## Setup

1. Open project in Android Studio
2. Sync Gradle
3. Configure Health Connect permissions
4. Update API base URL in `NetworkModule.kt`
5. Build and run

## Features

- Onboarding & Authentication
- Home dashboard
- Metrics trends
- Coach feed
- Visit requests
- Messaging
- Health Connect integration
