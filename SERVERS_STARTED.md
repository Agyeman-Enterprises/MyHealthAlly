# Servers Started

## Status

Starting all required services:

1. **Web Dashboard** - Starting on port 3001
2. **Backend API** - Starting on port 3000
3. **Prisma Studio** - Can be started with `pnpm prisma studio` in backend directory

## Access

Once started:
- Web: http://localhost:3001/login
- Backend: http://localhost:3000/health
- Database: http://localhost:5555 (when Prisma Studio is started)

Servers are starting in separate PowerShell windows.
